# task-manager-Web app

A simple web application that allows users to manage daily tasks. Users can sign up, log in, and add tasks which are stored in Firebase Firestore.

## Live Demo
https://task-manager-app-35a39.web.app

## Features
- User authentication (Signup / Login)
- Add tasks
- View tasks
- Delete tasks
- Data stored in Firebase Firestore
- Hosted using Firebase Hosting

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Firebase Authentication
- Firebase Firestore
- Firebase Hosting

## Project Structure

task-manager/
